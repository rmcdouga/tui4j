# Simple Example

The `simple` example demonstrates the simplest possible Bubble Tea application with a countdown timer.

<a href="./src/main/java/com/williamcallahan/tui4j/examples/simple/SimpleExample.java">
  <img width="750" src="../../assets/simple-tape.gif" alt="Animation showing TUI4J simple countdown example" />
</a>

## Features

- Minimal Bubble Tea application
- Basic key handling (q, ctrl+c to quit)
- Simple countdown timer model
- Clean quit handling

## Running

```bash
# From project root
./gradlew examplesJar
java -cp examples/generic/tui4j-examples.jar com.williamcallahan.tui4j.examples.simple.SimpleExample
```
