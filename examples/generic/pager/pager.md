# Sample Content for Pager Example

This is a sample markdown file that can be displayed in the pager example.

## Usage

Replace this content with your own text, or place a file named `pager.md` in the current directory or `examples/generic/pager/` directory.

## Features

The pager example demonstrates:
- Viewport bubble usage
- Scrollable content
- Scroll percentage indicator
- Header and footer styling
- Mouse wheel support
- Full terminal with alt screen

## Navigation

- `↑` / `↓` : Scroll line by line
- `PgUp` / `PgDn` : Scroll page by page
- `ctrl+f` / `ctrl+b` : Half page up/down
- `Home` / `End` : Go to top/bottom
- `q` / `ctrl+c` / `esc` : Quit
